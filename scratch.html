Scratch Content

Name Assignment

Description

Created my name in a scratch project. I had to have it do three different actions. When the green flag was clicked it had to run and I added a way of resetting it when the space key was pressed.

Concepts Leardned

  • Event blocks - when green flag clicked.
  • Motion blocks - making the sprite move along the x and y axis, I also made them rotate and change size.
  • Loops - making the sprite repeat an action for a set amount of time or forever.
  • Sprites - used the library to find sprites that were the letters of my name.

Computer Buying Project

Description

Created a presentation in scratch that presented 4 different options for buying a computer. The computer was for and animation student who need a laptop under $1600. We ended up choosing a Dell, Razor, Lenovo and Mac as the options.

Concepts Leardned

  • Conditional statements - when a sprite was touching another sprite it would display information about the computer.
  • Changing backgrounds - The background would change when a preset button was pressed.
  • Sprites - created sprites and programmed one of them to move when the WASD keys were pressed.

Final Project

Description

I had to create a game for my final project in scratch. The game was a survival game set in the woods. There were different items required for you to survive, that you would have to gather, and there were also wild animals that you would have to fight off. Once you survived seven day you would win.

Concepts Leardned

  • Variables - I created variables that represented different items. They would decrease in value as time went on or increase if you gathered the materials.
  • Creating Blocks - I created blocks that were actions that happened more than once.
  • Broadcasting - I used broadcasts so sprites could communicate with one another.
